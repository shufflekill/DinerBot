DinerBot • Music & Downloads

Commands Guide

Use !play to stream music and queue songs. Use /ytmp3 to download a file when you have access.

Music Commands

These commands control playback in voice channels.
!play / !p
Searches YouTube or accepts a URL. If something is already playing, the song is queued.
!queue
Shows the current queue and what is playing now.
!nowplaying / !np
Shows the current track and its link.
!clearqueue
Clears the upcoming queue.
!remove <index>
Removes a queued song by its number.
!skip
Skips the current song and moves to the next queued item.
!pause
Pauses playback.
!resume
Resumes playback.
!stop
Stops playback, clears the queue, and disconnects the bot.

Download Commands

These commands are for MP3 downloads and access control.
/ytmp3
Downloads a YouTube video as MP3. Access is controlled by the server role set by staff.
/setytmp3role
Sets which role can use /ytmp3. Requires Manage Server.
/clearytmp3role
Removes the role restriction for /ytmp3. Requires Manage Server.

Admin Notes

Access can be managed without editing code.
Server setting

Choose the role that can use /ytmp3.

Automatic

The bot remembers the role per server in config.json.

Safe fallback

If no role is set, the command can be left open or restricted by staff.

Permissions

Server staff can update access whenever needed.

Quick Tips

  • Join a voice channel before using !play.
  • If a song is already playing, !play adds it to the queue instead of interrupting playback.
  • If you cannot use /ytmp3, ask a server admin to set the allowed role.
